# Personal Website Starter

## Run locally

From `d:\codex\website`:

```powershell
python -m http.server 8080
```

Then open: <http://localhost:8080>

## What to edit first

- `index.html`: bio, experience, contact links
- `script.js`: project cards
- `styles.css`: colors, fonts, layout, motion

## Fast deploy options

- GitHub Pages
- Netlify (drag-and-drop `website` folder)
- Vercel (import repo)
